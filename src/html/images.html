Images

All images should be place inside figures. Use jpg, gif, or link from a CMS to the image as your image source. The following classes are a layout starting point that can be tweaked by adding an id or changed throughout the article by changing the class. For more options place a image inside a figure inside a column using the bootstrap grid system.

Inline image

All figures nested inside the inline column will display inline by default. 

<div class="inline>
	<figure>
		<img src="..." >
		<figcapti	on>...</figcaption> 
	</figure>
</div >

Inline-left

The inline-left class applied to a figure creates an image 50% the width of the inline column aligned left. If there is nothing to the right of the figure, content lower on the page may wrap up and fill the blank space.

<figure class="inline-left">
	<img src="...">
	<figcaption>...</figcaption>
</figure>

narrow and centering

The figure class narrow will shrink the figure to 25% width. The class center-block will center align to the element it is nested in.

<figure class="narrow center-block">>
	<img src="...">
	<...>
</div>

Wide

Apply the wide class to a figure to make it 150% width of its container. The figure below is contained in an inline class, so it is 150% the width of the main article.

<figure class="wide">
	<img src="...">
	<figcaption>...</figcaption>
</figure>

<figure class="wide-left">
	<img src="...">

Break

Make an image 100% of the screen width by placing a figure inside a row without a column. If you want the height 100% crop the orginal image or hardcode a height value. If you want the figcaption to allign with the article column place it within an inline div

<div class="row>
	...
</div>

<div class="row">
	<figure>
		<img src="...">
		<div class="inline">
			<figcaption>...</figcaption>
		</div>
	</figure>
</div>

Break-left

Apply the break-left class to a row containing an figure column and a caption column. Size use bootstrap grids.

<div class="row break-left">
	<div class="col-xs-9>
		<figure>
			<img src="...">
		</figure>
	</div>

	<div class="col-xs-9 col-md-3>
		<figcaption>....</figcaption>
	</div>
</div>

Break-right

Apply the break-right class to a row containing an figure column and a caption column. Size using bootstrap grids. Put a figcaption with the classes visible-xs visible-sm to the figure as well as a caption column with the classes hidden-xs and hidden-sm

<div class="row break-left">
	<div class="col-xs-9 col-xs-push-3>
		<figure>
			<img src="...">
			<figcaption class="visible-xs visible-xs">...</figcaption>
		</figure>
	</div>

	<div class="col-sm-3 col-sm-pull-9>
		<figcaption class="hidden-xs hidden-sm>....</figcaption>
	</div>
</div>

Carasoul

Carousels are a bootstrap element. They can be used inline, wide, custom grid structure, but does not work in break. Every caraousel needs a unique id for the wrapping div and controls.

<div id="carousel-example-generic" class="carousel slide" data-ride="carousel">
  <!-- Indicators -->
  <ol class="carousel-indicators">
    <li data-target="#carousel-example-generic" data-slide-to="0" class="active"></li>
    <li data-target="#carousel-example-generic" data-slide-to="1"></li>
    <li data-target="#carousel-example-generic" data-slide-to="2"></li>
  </ol>

  <!-- Wrapper for slides -->
  <div class="carousel-inner" role="listbox">
    <div class="item active">
      <img src="..." alt="...">
      <div class="carousel-caption">
        ...
      </div>
    </div>
    <div class="item">
      <img src="..." alt="...">
      <div class="carousel-caption">
        ...
      </div>
    </div>
    ...
  </div>

  <!-- Controls -->
  <a class="left carousel-control" href="#carousel-example-generic" role="button" data-slide="prev">
    <span class="glyphicon glyphicon-chevron-left" aria-hidden="true"></span>
    <span class="sr-only">Previous</span>
  </a>
  <a class="right carousel-control" href="#carousel-example-generic" role="button" data-slide="next">
    <span class="glyphicon glyphicon-chevron-right" aria-hidden="true"></span>
    <span class="sr-only">Next</span>
  </a>
</div>

grids

Use the bootstrap grid system for a custom layout. To remove the padding caused by nested columns, wrap the columns in a row grid 

<div class="row grid">
	<div class="col-xs-6">
		<figure>
			<img src="..." alt="">
			<figcaption>...</figcaption> 
		</figure>
	</div>

	<div class="col-xs-6">
		<figure>
			<img src="..." alt="">
			<figcaption>...</figcaption> 
		</figure>
	</div>
</div>

Change the column width to adjust the number of elements per row.

<div class="row grid">        
  <div class="col-xs-4">
    <figure>
      <img src="..." alt="">
    </figure>
  </div>

  <div class="col-xs-4">
    <figure>
      <img src="..." alt="">
    </figure>
  </div>

  <div class="col-xs-4">
    <figure>
      <img src="..." alt="">
    </figure>
  </div>

  <figcaption>Or a single caption</figcaption>
</div>
